For my local Dev-setup on Windows I will use the newly free RHEL distro and install it on WSL2.

First I updated WSL: wsl --update

Then I download RHEL at no cost: 
https://developers.redhat.com/content-gateway/file/rhel/Red_Hat_Enterprise_Linux_10.0/rhel-10.0-x86_64-boot.iso

I create the folder C:\Users\peter\wslDistroStorage

I import RHEL into the WSL:
wsl --import RHEL C:\Users\peter\wslDistroStorage\RHEL .\rhel-10.0-x86_64-wsl2.tar.gz

I check if it worked and is available:
wsl.exe --list

And start it:
wsl -d RHEL

Then i replcae the default root user with a user peter like in:
https://learn.microsoft.com/en-us/windows/wsl/use-custom-distro

Then I create a easy to use shortcut to open my RHEL distro with a simple click. 

The shortcut destination is: C:\Windows\System32\wsl.exe -d RHEL -u peter

In the location: C:\Users\peter\Documents\projects


Now I'm gonna start with installing packages like docker following the docker instructions:
https://docs.docker.com/engine/install/rhel/#install-using-the-repository

When su as user peter fails, just open wsl in the cli as root: wsl -d RHEL -u root

verify docker is up, running and working:  sudo docker run hello-world


Now i have to allow non-privileged users to run it in rootless mode.:)
