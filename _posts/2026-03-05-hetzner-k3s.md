The repo [hetzner-k3s](https://hetzner-k3s.com/) allows to deploy dev- and production ready k3s clusters on Hetzner. 
Hetzner gives you the same as AWS/azureGC but is up to 80% cheaoper than those.

Edit the default config and run "hetzner-k3s create --config cluster.yaml"

In a few minutes all the VMs are running and reachable with the local kubectl and the new kubecontext.
